Robust Malicious Domain Detection Based on Spatio-Temporal Hypergraph Networks

Introduce

Malicious domains are an important resource for adversaries to launch cyber attacks, making them key clues for network intrusion detection. In the real world, requests for various malicious domains mix together in the DNS traffic, which is a challenge for a traditional machine-learning model to achieve satisfactory detection performance. To solve the above challenge, we collect a substantial volume of DNS traffic for 12 months from a real large-scale network with 1 million users, in which we extract numerous requested domains, involving 267 attacks exploiting malicious domain names. Moreover, we measure and analyze the domain property differences of different attacks in terms of name semantics, registration information, and resolution records. These differences lead to the performance fluctuations of machine learning algorithms when detecting various malicious domains. Therefore, we introduce a spatiotemporal hypergraph networks model, building high-order associations among domain properties to enhance the generalization ability and robustness of the detection model. Specifically, we introduce spatio-temporal correlation analysis into the hypergraph network to model the similarity of name semantics, domain attributes and request behaviors of malicious domains. Next, we apply reinforcement self-learning to improve the detection performance for some under-sampled attack types. Finally, extensive testing experiment results show that our model can achieve an impressive 97% precision and 98% recall on average. Furthermore, we integrate the model into a malicious domain detection system that can achieve 96.3% precision and 96.2% recall on real traffic for three months.
